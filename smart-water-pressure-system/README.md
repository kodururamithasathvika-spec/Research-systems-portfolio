# Smart Water Pressure Monitoring & Leak Alert System

## Objective
Develop a real-time system to monitor water pipeline pressure and detect abnormal drops or spikes indicating leaks.

## System Design
Pressure sensors integrated with a microcontroller and GUI-based visualization to track live pressure data and trigger alerts.

## Application Scope
Smart homes, municipal water distribution systems, and industrial pipeline monitoring.
