This project presents a wearable assistive hand gesture glove designed to

communication assistance. The system targets simple Humanâ€“Computer Interaction
(HCI) scenarios where verbal communication may be limited.

The focus of this project is on hardware integration, sensor interfacing, and
real-time system behavior rather than algorithmic novelty or software
optimization.

---

### System Description
- Flex sensors mounted on fingers for gesture capture
- Embedded microcontroller for real-time signal interpretation
- Predefined gesture-to-text mapping
- LCD-based textual output for immediate feedback
